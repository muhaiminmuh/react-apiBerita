# React Native - API Berita


# Requirement

* `JDK 8`
* `SDK`
* `nodeJS-v8.12.0`
* `react-native@0.55.4`
* `react-navigation`

# Installation

1. Download project ini, Copy ke folder master React Native kalian
2. Gunakan API yang disediakan sebagai Web Service 
3. Setting URL di `halaman/berita.js`
